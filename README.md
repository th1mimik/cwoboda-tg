#Руководство cwoboda-tg
##Установка:
1. Установите Termux из Google Play или F-Droid.
2. В приложении пишите:
```
pkg install python
```
На все вопросы "... Y/n" выбираете "Y".
Проверяете, что python скачался:
```
python --version
```
Если будет _"Python 3.x.x"_ то все скачалось (x - число).
3 Устанавливаем сам файл:
```
pkg install git
```
```
git clone https://github.com/th1mimik/cwoboda-tg.git
```
Проверяете, что все сделано правильно:
```
ls
```
В консоли появится синим текстом слово _"cwoboda-tg"_.
Пишем слудующие команды:
```
cd cwoboda-tg
```
```
pip install pycryptodome
```
Убедитесь, что **pycryptodome** скачался без ошибок:
```
pip show pycryptodome
```
Если нету красного текста то все в норме.
##Запуск:
1. Пишите в консоли:
```
python run.py
```
Если ошибок нету появится текст, что программа работает.
##Настройка в телеграме:
1. Заходите в настройки.
2. Переходите в раздел "ДАННЫЕ И ПАМЯТЬ".
3. Листаете в самый низ и нажимаете на "НАСТРОЙКА ПРКОСИ".
4. Листаем вниз и нажимаем на "ДОБАВИТЬ ПРКОСИ".
5. В строке **"СЕРВЕР"** пишите _"127.0.0.1"_ либо же заходите в Termux и смотрите в консоли строку _"cwoboda-tg -> proxy x.x.x.x"_ копируем эти цифры, главное до **":"** пример: _127.0.0.1_
6. В настройках прокси на строке **"ПОРТ"** пишем то, что после **":"** в консоли Termux, пример: _1080_
7. Если все сделанно **правильно** то ошибок подключения не будет, могут быть небольшие перепдключения.
